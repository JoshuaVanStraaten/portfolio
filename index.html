Projects

AI Web Scraper - Python

Through the integration of local Ollama language models, selenium and streamlit - for a user interface - I developed a web scraper that can extract data from any website. The scraper is capable of parsing HTML and JSON data, and it can be customized to scrape specific elements or attributes and return the data in a structured format. The project is designed to be user-friendly, allowing users to input the URL of the website they want to scrape and select the specific data they want to extract.

Firmware Applications - Embedded C

Throughout the design lifecyle of a microchip based system, I have developed firmware applications for RISC-V architecture. These applications include the use of various peripherals such as ADCs, timers and UART. These applications are designed to be modular and reusable, allowing for easy integration into different projects. The firmware is written in C and is optimized for performance and memory usage. The applications are tested and validated to ensure they meet the required specifications and performance standards.

Business Website - HTML, CSS, JS
